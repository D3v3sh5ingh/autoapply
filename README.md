# AutoApply + JobAnalytics

An open-source, commercially-friendly job application agent that helps you find and apply to jobs intelligently.

## Features (Planned)
- **Scraper**: Automated job gathering from various boards.
- **Matcher**: Resume vs. Job description matching to rank opportunities.
- **Analytics**: Dashboard to track applications and match rates.
- **Auto-Apply**: Optional automated application submission.

## Setup

1. **Clone the repository**
   ```bash
   git clone <repository_url>
   cd autoapply
   ```

2. **Create a virtual environment**
   ```bash
   python -m venv venv
   source venv/bin/activate  # On Windows: venv\Scripts\activate
   ```

3. **Install dependencies**
   ```bash
   pip install -r requirements.txt
   ```
